# Exoplanet-Habitability-Detection.
This is a program built to search for designated exoplanets from specific space missions and determine whether they exoplanet lies within the habitable zone of it's star or not.
The notebook organizes and takes user input for the star's name and the mission name.
It also takes the minimum and maximum limit for orbital days as user input.
The orbital period is then exported to a .csv file which is used inside the calculation program.
The calculation enables us to determine whether the planet lies within the habitable zone of its star or not, by taking basic user defined parameters.
It works on known and scientifically proven formulae to determine the habitability.
Once it returns that the planet lies within the habitable zone, it deploys an interactive gaseous component analysis of the planet's atmosphere.
User input then determines whether the planet is habitable, partially habitable, or rather not suitable for habitation.
